Join the Community Pilot

We aren't in the Chrome Web Store just yet! As a pilot user, you can install the extension manually by following these simple steps.

Download Pilot Version (ZIP)

  • 1. Unzip the File

    Find the downloaded ZIP file on your computer. Right-click it and select "Extract All". You need the folder inside, not the ZIP file itself.

  • 2. Open Extension Management

    In Chrome, copy and paste this address into a new tab and press enter:

    chrome://extensions

  • 3. Enable Developer Mode

    In the top right corner of the extensions page, toggle the "Developer mode" switch to ON.

  • 4. Load the Folder

    Click the new "Load unpacked" button that appeared. Select the folder you unzipped in Step 1. That's it!
